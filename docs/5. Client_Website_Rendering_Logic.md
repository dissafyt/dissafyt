# Website Rendering Logic

The platform dynamically serves client websites.

Step 1

User visits a domain or subdomain.

Example:

barber.dissafyt.co.za

---

Step 2

System reads request host header.

request.get_host()

---

Step 3

System finds matching Website record.

---

Step 4

Website template is loaded.

Example:

sites/barber_template/home.html

---

Step 5

CMS data is injected.

Example context:

business
services
testimonials
gallery

---

Step 6

Django renders the template and returns HTML.

---

Result

A complete website is served from the Dissafyt platform.
