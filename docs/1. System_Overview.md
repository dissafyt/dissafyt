# Dissafyt Platform – System Overview

## Purpose

Dissafyt is a Website-as-a-Service platform designed to generate and host client websites using a centralized Django rendering engine.

The system enables service businesses to quickly launch professional websites using pre-built templates populated with business data collected through an onboarding process.

The platform is composed of three primary layers:

1. Marketing Website
2. Client Dashboard
3. Client Website Rendering Engine

All websites are dynamically rendered using Django templates and hosted within the same platform infrastructure.

---

## Platform Structure

Visitor → Marketing Website → Client Signup → Dashboard Onboarding → CMS Data → Template Rendering → Website Live

Example website URL:

example.dissafyt.co.za

Later connected to:

examplebusiness.co.za

---

## Core Principles

• One centralized Django backend  
• One codebase hosting many client websites  
• Templates + CMS data = generated website  
• Subdomain first, custom domain later  
• No external builders required  

---

## Target Users

Small businesses needing affordable websites.

Examples:

• Plumbers  
• Barbers  
• Restaurants  
• Local service providers  

---

## MVP Goals

1. Capture leads from marketing site
2. Allow client onboarding
3. Store business data
4. Select template
5. Generate website instance
6. Publish to subdomain
