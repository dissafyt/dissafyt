# Core Data Model

## BusinessProfile

Represents the client's business.

Fields:

• owner (User)
• business_name
• phone
• email
• address
• description
• logo

---

## Website

Represents a generated website.

Fields:

• owner
• business_profile
• template_name
• subdomain
• custom_domain
• published

---

## Service

Represents a service offered by the business.

Fields:

• website
• name
• description

---

## Testimonial

Fields:

• website
• author
• quote

---

## GalleryImage

Fields:

• website
• image
• caption
